### KKP Workshop on Science Reproducibility - Gödöllő - 2026.04.29 (Wednesday)

**Title:** From Script to Document: Reproducible Research with Quarto

**Description:**
Are you tired of copy-pasting tables and figures from R into Word? In this
workshop, you will learn how to use Quarto to create fully reproducible
documents that integrate your analysis, visualizations, and text.
We will introduce Quarto as an open-source scientific publishing system for
reproducible research. Participants will learn how to combine R code, data
wrangling, statistical output, and narrative text into a single document.
From one file to many outputs — HTML, PDF, Word, slides, and more.
We will also demonstrate practical tips such as automatic model equations
and publication-ready tables. Quarto also supports Python and other languages.
No prior experience with Quarto or R Markdown is needed — just basic
R/RStudio familiarity.

**Duration:** ~1 hour

**Requirements:** Laptop with R, RStudio, and Quarto installed (instructions
will be provided in advance)
